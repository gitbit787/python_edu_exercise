# Python Education Repository

This repository is a structured, multi-level Python learning resource designed for **beginners**, **intermediate**, and **advanced** learners.

It combines:
- **Concept explanations** in plain language
- **Runnable example code**
- **Exercises with tests and solutions**
- **Small projects** at different difficulty levels

## Repository structure

- `beginner/` – Core Python syntax and fundamentals
- `intermediate/` – Data structures, modules, OOP, error handling, testing
- `advanced/` – Generators, decorators, typing, async, performance
- `projects/` – Small applied projects at multiple levels
- `exercises/` – Standalone problem sets with tests
- `docs/` – Curriculum maps, teaching guides, and roadmap

## How to use this repo

1. **Pick your level**
   - New to programming: start in `beginner/` and follow the numbered folders.
   - Have some Python: skim `beginner/`, then move into `intermediate/`.
   - Comfortable with Python basics: explore `advanced/` topics and projects.

2. **Run the examples and tests**
   - Create a virtual environment:
     - `python -m venv .venv`
     - Activate it (PowerShell): `.\.venv\Scripts\Activate.ps1`
   - Install dependencies:
     - `pip install -r requirements.txt`
   - Run tests (once they are added):
     - `pytest`

3. **Practice deliberately**
   - Try exercises before looking at solutions.
   - Use the tests to check your work.
   - Experiment by modifying examples and observing the results.
