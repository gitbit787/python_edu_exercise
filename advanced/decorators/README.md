## Advanced – Decorators

**Goal**: Understand and implement function decorators in Python.

### Topics covered

- Functions as first-class objects
- Writing simple decorators
- Using `functools.wraps`
- Practical use cases (logging, timing, access control)

### Files

- `lesson.py` – annotated examples of writing and using decorators.
- `exercises.py` – tasks where you implement your own decorators.
- `solutions.py` – reference implementations.
