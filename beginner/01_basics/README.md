## 01 – Python Basics

**Goal**: Get comfortable running Python, using `print`, comments, and basic data types.

### What you will learn

- How to run a Python script
- Using `print` to display output
- Writing comments
- Working with basic types: integers, floats, strings, and booleans

### Files in this folder

- `lesson.py` – runnable examples that you can read and execute.
- `exercises.py` – functions with `TODO` markers for you to implement.
- `solutions.py` – one possible set of solutions. Try the exercises first!

### Suggested workflow

1. Read through `lesson.py` and run it.
2. Open `exercises.py` and implement each function.
3. Compare your solutions with `solutions.py`.
4. Experiment: change values and add your own examples.
