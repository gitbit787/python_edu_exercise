## 02 – Variables and Types

**Goal**: Learn how to store values in variables and work with core Python data types.

### What you will learn

- Naming and assigning variables
- Reassigning and updating values
- Working with numbers, strings, and booleans
- Converting between types (e.g. string to int)

### Files in this folder

- `lesson.py` – examples that demonstrate variables and types in context.
- `exercises.py` – functions for you to implement using variables and type operations.
- `solutions.py` – reference implementations to compare with your own.

### Suggested workflow

1. Read and run `lesson.py` to see the concepts in action.
2. Implement the functions in `exercises.py`.
3. Compare with `solutions.py` and reflect on any differences.
