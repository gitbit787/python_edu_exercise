## Curriculum Map

This document will outline the recommended learning paths through the repository.

- Beginner path:
  - Start in `beginner/01_basics/`
  - Progress through the numbered beginner folders
  - Attempt at least one project in `projects/beginner/`

- Intermediate path:
  - Review any weak areas in `beginner/`
  - Work through `intermediate/` topics in order
  - Attempt projects in `projects/intermediate/`

- Advanced path:
  - Focus on specific topics in `advanced/` that match your goals
  - Tackle challenging exercises and projects in `projects/advanced/`

This file will be expanded as more lessons and projects are added.
