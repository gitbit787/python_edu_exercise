## Intermediate – Data Structures (Deeper)

**Goal**: Go beyond the basics of lists and dictionaries and learn when and why to use different data structures.

### Topics covered

- Lists vs tuples vs sets vs dictionaries
- Common operations and performance considerations
- The `collections` module (`deque`, `Counter`, `defaultdict`, etc.)

### Files

- `lesson.py` – worked examples demonstrating these ideas.
- `exercises.py` – practice problems using real-world style data.
- `solutions.py` – reference implementations.
