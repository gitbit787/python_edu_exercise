## Beginner Project – Number Guessing Game

**Goal**: Practice basic input/output, control flow, and loops by building a simple number guessing game.

### Description

The program should:

- Pick a random secret number within a range (e.g. 1–100).
- Ask the user to guess the number.
- Tell the user whether each guess is too high, too low, or correct.
- Limit the number of guesses, or allow unlimited guesses until correct.

### Structure

- `starter/` – starter implementation with TODOs.
- `solution/` – one complete reference implementation.

### How to run

From the project root:

- `python projects/beginner/number_guessing_game/starter/game.py`
